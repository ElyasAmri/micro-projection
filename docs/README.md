# Documentation

Project documentation for the micro-projection monorepo (see the
[top-level README](../README.md) for an overview of the `app/` and `simulation/`
subprojects).

## Conventions

- Tracked docs live directly under `docs/` (committed to git).
- Personal, draft, or non-shareable material goes in **`docs/.local/`**, which is
  git-ignored. More generally, any path whose name contains `.local` is ignored
  (the `*.local*` rule in the root `.gitignore`), so a quick scratch note can be
  named e.g. `notes.local.md` and stay out of version control.
